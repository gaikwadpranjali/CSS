This is Eaxmple of Inline CSS


Inline CSS is a method used to apply unique styles directly to a single HTML element by using the style attribute.

An inline CSS is used to apply a unique style to a single HTML element.
An inline CSS uses the style attribute of an HTML element.

    Key points:

  • Applies styles to a single element.
  • Uses the style attribute.
  • Has higher priority than internal and external CSS.
  • Easy for small changes.
  • Difficult to maintain in large projects.
  • Best used sparingly, while external CSS is preferred for most professional websites.
